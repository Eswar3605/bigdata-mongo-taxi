# Dashboard Screenshots

This directory contains screenshots of the Streamlit dashboard visualizations.

## Required Screenshots

Please add the following screenshot files to this directory:

1. **daily_revenue_trips.png** - Daily Revenue & Trips line chart with date range slider
2. **top_pickup_zones.png** - Top Pickup Zones bar chart with interactive slider
3. **payment_breakdown.png** - Payment Breakdown bar chart and table

These screenshots are referenced in the main README.md file and demonstrate the three required visualizations for the project.

## How to Capture Screenshots

1. Run the dashboard: `uv run streamlit run bigdata_mongo_taxi/viz/dashboard.py`
2. Navigate to `http://localhost:8501` in your browser
3. Take screenshots of each visualization section
4. Save them with the filenames listed above in PNG format
